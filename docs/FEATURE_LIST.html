客戶管理 Phase 1

  • 客戶 CRUD - 編號 / 姓名 / 電話 / 來源 / 備註 / 加入日期
  • 多地址（同一客戶可存多個地址，標籤化）
  • 多機型（一客戶多台機器，類型 / 廠牌 / 型號 / 備註）
  • 全域模糊搜尋（姓名 / 電話 / 編號 / 備註）
  • 縣市 + 鄉鎮二段下拉篩選（台中 / 彰化 / 南投 / 雲林共約 90 個鄉鎮）
  • 客戶列表 KPI（總客戶數、本月活躍、即將到期）
  • 客戶列表 CSV 匯出（UTF-8 BOM，Excel 直開）
  • 客戶詳情 - 累計消費 / 服務次數 / 距上次服務 / 平均週期 / 臨時取消次數
  • 服務歷史列表（含取消原因顯示）
  • 一鍵新增訂單按鈕（自動帶客戶 ID）
  • 服務歷史 hover「複製此單」按鈕

訂單管理 Phase 2

  • 訂單 CRUD - 客戶 / 地址 / 時間 / 狀態 / 收款方式
  • 多服務項目（一單可多項，各項可指派不同師傅）
  • 多加減項（加大 / 其他加價 / 折扣，可自訂名稱金額）
  • 訂單編號自動產生 YYYYMMDD-NNN（每日重置流水號）
  • 應收總額即時試算（含加減項）
  • 時長輸入（分鐘）取代結束時間 picker - 快選 60 / 90 / 120 / 180 chips
  • 選客戶後自動帶出該客戶地址與機器列表
  • 服務項目選擇後自動帶出預設價
  • 訂單列表 - 廣義搜尋 + 狀態 / 收款 / 回繳三維篩選
  • 訂單列表 KPI（筆數 / 總金額 / 已完成金額）
  • 訂單詳情 - 完整資訊 + 狀態 badges + 金額分解
  • 訂單一鍵複製為新訂單（保留服務內容，清空日期）
  • 訂單快速取消（月曆 × 按鈕，prompt 原因記錄）
  • 取消原因可見化（訂單詳情 alert + 客戶歷史顯示）

收款 / 回繳雙欄位機制 Phase 2

  • 收款方式 - 未收款 / 已收款-現金 / 匯款 / 刷卡 / LINE Pay
  • 回繳狀態 - 待回繳 / 已回繳 / 免回繳
  • DB trigger 自動連動（匯款類別 → 自動免回繳）
  • 師傅待回繳總覽頁 - 按師傅分卡顯示待回繳總額 + 訂單明細
  • 一鍵「全部標記已回繳」批次核帳
  • 師傅 PWA「我收到現金了」一鍵切收款方式
  • 師傅 PWA 自己 banner 顯示累計待回繳金額

月曆排案 Phase 2

  • FullCalendar 月 / 週視圖切換
  • 師傅切換 tabs（全部 / 各師傅）+ 左右箭頭循環
  • 每位師傅獨立顏色，案件 event 用對應色塊
  • 左側待派工面板 - 完整地址 + 服務摘要
  • 外部拖拉派工 - 從待派工面板拖到月曆某日，自動排程 + 自動指派該師傅
  • 月曆 event 可拖拉改日期
  • Event 顯示時間區段（10:00 - 11:30）
  • Hover tooltip 顯示完整資訊（訂單編號 / 客戶 / 電話 / 地址 / 師傅 / 服務 / 金額）
  • 每日總額 badge（日格右上角顯示營業額）
  • 快速取消按鈕（hover 顯示 ×）
  • 點空白日跳新訂單表單（帶日期）

師傅 PWA Phase 2

  • 師傅獨立帳號 + 手機優化介面
  • RLS 限制只看自己被指派的案件
  • 今日案件清單（前後天切換）
  • 案件詳情 - 客戶 / 地址 / 電話一鍵撥打 / Google Maps 一鍵導航
  • 顯示備註提醒（住二樓 / 養狗等特殊狀況）
  • 三大現場按鈕 - 我收到現金了 / 客戶說已匯款 / 完成此案件
  • 待回繳金額 banner（自己監控）
  • 我的本月薪資頁 - 每日明細 + 加減項 + Excel 匯出
  • 底部 tab 導航（今日案件 / 我的薪資）
  • PWA 可加到主畫面（manifest + service worker）

儀表板與報表 Phase 3

  • 儀表板 5 個 KPI - 本月營業額 / 案件數 / 今日待完成 / 即將到期 / 本月取消率
  • 營業額月對比（vs 上月 ↑↓ 百分比）
  • 取消率 ≥ 20% 紅色警示
  • 師傅本月案量直條圖（已完成綠 / 待完成橘）
  • 儀表板今日案件清單（時間 / 客戶 / 區域 / 師傅）
  • 師傅待回繳一鍵跳轉按鈕
  • 營業報表 - 期間切換（今日 / 本週 / 本月 / 本季 / 今年 / 自訂日期）
  • 4 個 KPI - 期間營業額 / 完成件數 / 服務客戶 / 平均客單價
  • 服務項目 TOP（含金額累計）
  • 區域熱區（前 10 縣市鄉鎮）
  • 付款方式分佈（百分比 + 金額）
  • 師傅營業額排行
  • VIP 客戶 TOP 10（可點擊跳客戶詳情）
  • 營業報表 CSV 匯出（UTF-8 BOM）

計件薪資 Phase 3

  • 師傅月報入口（各師傅卡片）
  • 月報視圖 - 重現「2026年五月份」Excel 版型
  • 每日明細 - 日期 / 當日應收 / 4 台展開 / 加大 / 折扣 / 匯款
  • 本月應收（含加減項）+ 件數合計
  • 月份前後切換
  • Excel 匯出 .xlsx（SheetJS）
  • 師傅自己可在 PWA 查看本月薪資

即將到期客戶 Phase 3

  • 11-13 個月未服務客戶自動偵測（pg_cron 已部署但需手動觸發）
  • 狀態 tabs - 待通知 / 已通知 / 已跳過
  • 逾期天數 badge（逾期 N 天 / 本週到期 / N 天後到期）
  • 客戶詳情 + 上次服務日期
  • 一鍵複製 LINE 提醒文案（正體中文範本）
  • 「已通知」記錄誰何時通知
  • 「重新掃描」立即觸發 RPC 更新名單

系統設定 Phase 1

  • 帳號管理（僅 owner）- 新增 / 編輯 / 重設密碼 / 停用 / 刪除
  • 三種角色 - 老闆 / 管理員 / 師傅
  • 服務項目 CRUD（代碼 / 名稱 / 預設價 / 分類 / 排序）
  • 折扣 / 加價項目 CRUD
  • 客戶來源 CRUD
  • 操作稽核 /settings/audit - 最近 300 筆敏感操作（僅 owner）
  • 稽核覆蓋 - 取消訂單 / 刪除訂單 / 刪除客戶 / 批次回繳 / 帳號操作
